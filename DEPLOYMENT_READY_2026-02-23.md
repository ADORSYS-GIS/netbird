## USAGE

### First Time Deployment

```bash
cd infrastructure/ansible-stack

# 1. Prepare configuration
cp terraform.tfvars.example terraform.tfvars
vim terraform.tfvars  # Edit with your values

# 2. Initialize
terraform init

# 3. Plan
terraform plan -out=tfplan

# 4. Apply
terraform apply tfplan

# 5. Verify (wait 2-3 minutes for services to start)
terraform output -raw management_server_ip | xargs -I {} curl https://{}/health
```

### Subsequent Updates

```bash
# Update configuration
vim terraform.tfvars

# Plan changes
terraform plan

# Apply
terraform apply

# Terraform handles:
# - Creating new resources
# - Re-running Ansible
# - Updating Docker containers
# - Zero-downtime rolling updates (if HA enabled)
```

### Complete Cleanup

```bash
terraform destroy

# This will:
# 1. Run cleanup playbook
# 2. Stop all Docker containers (docker-compose down -v)
# 3. Remove cloud resources
# 4. Delete terraform state
```
