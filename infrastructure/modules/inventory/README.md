# Inventory Module

Consolidated inventory module supporting multiple cloud providers (AWS, GCP, Azure) and manual hosts.

## Submodules

- `providers/aws`: AWS EC2 discovery via tags.
- `providers/gcp`: Google Compute Engine discovery via labels.
- `providers/azure`: Azure Virtual Machine discovery via tags.
- `providers/manual`: Static host definition.

## Usage

See specific provider READMEs for detailed configuration.
