# Helm Configuration & Documentation

This directory contains **Helm** related configurations for deploying NetBird on Kubernetes.

## Structure
- `charts/`: Local Helm chart overrides or custom charts (if any).
- `values/`: Environment-specific value files (e.g., `prod-values.yaml`).
- `docs/`: Helm-specific documentation (upgrade guides, value reference).

## Documentation
Please refer to the `docs/` directory for detailed usage instructions.
