# Ansible Configuration & Documentation

This directory contains all **Ansible** related code for deploying NetBird on Docker/VM environments.

## Structure
- `roles/`: Reusable Ansible roles for NetBird components (Management, Signal, etc.)
- `playbooks/`: Main playbooks to execute the roles.
- `inventory/`: Dynamic or static inventory files.
- `docs/`: Ansible-specific documentation (guides, variable reference).

## Documentation
Please refer to the `docs/` directory for detailed usage instructions.
