# Ansible Configuration & Ansible Configuration

Automated NetBird deployment to VMs discovered by Terraform.

## Structure

```
configuration/ansible/
├── ansible.cfg              # Ansible configuration
├── site.yaml                # Main deployment playbook
├── inventory/
│   ├── terraform_inventory.yaml  # Generated by Terraform
│   └── group_vars/all.yml   # Variables and secrets
└── roles/
    ├── common/              # System hardening (firewall, SSH, NTP)
    ├── docker/              # Docker installation
    └── netbird/             # NetBird deployment
```

## Quick Start

```bash
# 1. Configure variables
vim inventory/group_vars/all.yml

# 2. Encrypt secrets
ansible-vault encrypt inventory/group_vars/all.yml

# 3. Deploy
ansible-playbook site.yaml --ask-vault-pass
```
